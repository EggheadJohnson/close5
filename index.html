REST API Methods

Get methods

  • /date?sortOrder={oldest, newest} - return the items listed in date order either oldest first or newest first
    • Errors on:
      • sortOrder neither of "oldest" or "newest"
  • /price?sortOrder={highest, lowest} - return the items listed in price order either highest first or lowest first
    • Errors on:
      • sortOrder neither of "highest" or "lowest"
  • /item/{itemID} - return a single item with the ID sent or blank if no item was found with that ID
    • Errors on:
      • Item not found
  • /getByUser/{userID} - return a list of items that the user represented by userID is selling or an empty array if no items or no user is found
    • Errors on:
      • user not found or has nothing for sale
  • /getByLocation?lat={latitude}&lon={longitude} - return a list of items which can be found within 50 miles of the latitude and longitude passed in
    • Errors on:
      • latitude and/or longitude missing

Queries Available on All* End Points

*Except /item

  • ?limit={maximum} - limit the number of results returned
  • ?status={tos, removed} - limit the results returned to items with either a status of tos or removed
